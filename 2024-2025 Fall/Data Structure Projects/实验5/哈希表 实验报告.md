## 算法设计

用除留余数法构建的哈希函数为：
$$
H(key)=key \mod p\text{ },\text{ }p\leq m
$$
此题中直接取哈希表长m=p。

线性探测法求得的第i个地址为：
$$
H_i=(H(key)+i)\mod m \text{, }i=1,2,\dots,k(k\leq m-1)
$$
拉链法直接采用链表，在同一地址上存储多个记录。

查找时直接利用关键字key求出地址H(key)，在HT1上利用线性探测寻找该记录，成功查找次数$S$即为线性探测的次数i，若遇到空位置说明查找失败，失败查找次数$U$即为空位置与初始位置的距离；在HT2上遍历该地址上的链表，成功查找次数$S$即为该记录在链表上的位置，遍历完仍未找到说明查找失败，失败查找次数$U$即为链表长度+1。

假设各记录被查找概率相等即可计算出ASL：
$$
ASL = \frac{1}{n}\sum^{n}_{i=1}S_i
$$

### 数据结构

线性探测法构建的HT1结构定义如下：

```c++
typedef struct HTable{
	int *elem;  //存储元素
	int *successtime;  //存储每个元素的成功查找次数
	int *failtime;   //存储每个元素的失败查找次数
	int count;  //记录个数
}HashTable;
```

拉链法构建的HT2结构定义如下：

```c++
typedef struct CNode{
	int elem;
	struct CNode *next;
}CNode,*Chain;

typedef struct CHTable{
	Chain *ChainHash;
	int *ChainLength; //存储每个链表的长度
	int count; //记录个数
}CHashTable;
```

采用全局变量tablesize=p标记线性探测法中的p和表长m，全局变量num标记记录总数。

## 设计目标

在C语言环境下，设计一个稀疏一元多项式运算器，要求实现如下功能：

1. 输入并创建多项式（按指数升序或降序排列，系数浮点型，指数整型）

   输入浮点型系数和整型指数，建立多项式并存储。

2. 输出多项式，项数+每项系数指数（按指数升序或降序排列）

3. 求和：多项式相加

   输入加数多项式的序号，输出相加结果并询问是否储存。

4. 求差：多项式相减

   输入被减数与减数多项式的序号，输出相减结果并询问是否储存。

5. 求值

   输入多项式序号与浮点型数值，输出多项式在该数处的取值

6. 销毁

   输入多项式序号，删除多项式全部内容

7. 清空

   输入多项式序号，将多项式清零

8. 修改
   1. 插入新项：输入多项式序号、新项的系数与指数，将新项插入多项式
   2. 删除项：输入多项式序号、需删除项的指数，将该项从多项式中删除
   3. 修改已有项：输入多项式序号、需修改项的指数、修改后的系数与指数，修改该项

9. 微分（N阶导数）

   输入多项式序号和需求的微分阶数，输出微分结果并询问是否储存

10. 不定积分

    输入多项式序号，输出不定积分结果并询问是否储存

11. 定积分

    输入多项式序号与定积分起点、终点，输出定积分结果

12. 乘法和乘方
    1. 乘法：输入乘数多项式的序号，输出相乘结果并询问是否储存
    2. 乘方：输入乘方多项式的序号与乘方次数，输出乘方结果并询问是否储存

13. 带余除法   

    输入被除数与除数多项式的序号，输出商多项式与余数多项式，分别询问是否储存

## 算法设计

### 数据结构

采取菜单式驱动，循环输出菜单界面，输入操作序号，执行对应操作。

采用带头结点的链表存储多项式，链表类型定义如下：

```c
typedef struct LNode{
	double coef; //该项系数
	int exp;     //该项指数
	struct LNode *next;
}LNode,*LinkList;
```

使用链表头结点的exp变量存储链表长度。

采用链表指针数组 p[NUM] 存储各链表的头指针，数组元素初值为NULL，创建链表时从头开始查找空位置，将链表头指针存入第一个空位置中。

除法功能结果较为特殊，包含商多项式和余数多项式，故采用两个链表头结点组成的结构体类型存储除法的结果，类型定义如下：

```c
typedef struct DivResult{
	LinkList quotient;  //商多项式的头指针
	LinkList remainder; //余数多项式的头指针
}DivResult;
```

## 调试分析

各功能测试数据与问题解决如下：

1. 输入并创建多项式：

   1. 常规输入 1 2.5 3 6.1 8 -2.3 -1，没有出现问题。

   2. 输入负指数 -2 1 -1，问题：创建了含负指数结点。

      解决方法：在输入指数时添加合法性检测，只允许输入非负整数和-1（结束指令）。

      为插入、修改同样加入该检测。

   3. 输入重复指数 1 1 1 1 -1，问题：创建了指数重复的结点。

      解决方法：在输入指数后利用Locate函数检测是否重复，若重复则提示重新输入。

      为插入、修改同样加入该检测。

   4. 输入零系数 2 0 -1，问题：创建了零系数结点，空占位置。

      解决方法：在输入系数时检测是否为0，若为0则不创建结点。

      为插入、修改同样加入该检测。

2. 输出多项式：

   1. 常规输入，没有出现问题。

   2. 输入未创建的序号，问题：无法输出未创建的多项式。

      解决方法：为各功能输入序号时添加合法性检测，若当前位置元素为NULL，则提示未创建多项式。

   3. 输入仅含头结点多项式的序号，问题：无法输出仅含头结点的多项式。

      解决方法：在输出前增加判断，若仅含头结点，则输出0.000。

3. 求和：

   1. 常规输入，没有出现问题。

   2. 输入含有指数相同、系数互为相反数项的多项式，问题：结果出现零系数结点。

      解决方法：计算同指数项时检测系数结果是否为0，若为0则不创建结点。

      在求差中同样加入该检测。

4. 求差：没有出现问题。

5. 求值：没有出现问题。

6. 销毁：注意将数组p对应位置重置为NULL，没有出现问题。

7. 清空：没有出现问题。

8. 插入新项：已添加合法性检测，没有出现问题。

9. 删除项：利用Locate函数寻找删除项位置，该项不存在则打印未找到该项，没有出现问题。

10. 修改已有项：已添加合法性检测，没有出现问题。

11. 微分：

    1. 常规输入，没有出现问题。

    2. 输入负数阶数，问题：无法计算负阶数微分。

       解决方法：加入合法性检测。

       在乘方中同样加入该检测。

12. 不定积分：没有出现问题。

13. 定积分：没有出现问题。

14. 乘法：没有出现问题。

15. 乘方：没有出现问题。

16. 除法：

    1. 常规输入，问题：商多项式项数错误。

       解决方法：在计算过程中添加商多项式L->exp的自增。

    2. 被除数输入0，问题：结果错误。

       解决方法：加入合法性检测。

    3. 除数项数大于被除数项数，问题：结果错误。

       解决方法：加入合法性检测。

```c
	//空间复杂度：O(3a)
```

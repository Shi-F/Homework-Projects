# 稀疏一元多项式运算器 实验报告

## 设计目标

在C语言环境下，设计一个稀疏一元多项式运算器，要求实现如下功能：

1. 输入并创建多项式（按指数升序或降序排列，系数浮点型，指数整型）

   输入浮点型系数和整型指数，建立多项式并存储。

2. 输出多项式，项数+每项系数指数（按指数升序或降序排列）

3. 求和：多项式相加

   输入加数多项式的序号，输出相加结果并询问是否储存。

4. 求差：多项式相减

   输入被减数与减数多项式的序号，输出相减结果并询问是否储存。

5. 求值

   输入多项式序号与浮点型数值，输出多项式在该数处的取值

6. 销毁

   输入多项式序号，删除多项式全部内容

7. 清空

   输入多项式序号，将多项式清零

8. 修改
   1. 插入新项：输入多项式序号、新项的系数与指数，将新项插入多项式
   2. 删除项：输入多项式序号、需删除项的指数，将该项从多项式中删除
   3. 修改已有项：输入多项式序号、需修改项的指数、修改后的系数与指数，修改该项

9. 微分（N阶导数）

   输入多项式序号和需求的微分阶数，输出微分结果并询问是否储存

10. 不定积分

    输入多项式序号，输出不定积分结果并询问是否储存

11. 定积分

    输入多项式序号与定积分起点、终点，输出定积分结果

12. 乘法和乘方
    1. 乘法：输入乘数多项式的序号，输出相乘结果并询问是否储存
    2. 乘方：输入乘方多项式的序号与乘方次数，输出乘方结果并询问是否储存

13. 带余除法   

    输入被除数与除数多项式的序号，输出商多项式与余数多项式，分别询问是否储存

## 算法设计

### 数据结构

采取菜单式驱动，循环输出菜单界面，输入操作序号，执行对应操作。

采用带头结点的链表存储多项式，链表类型定义如下：

```c
typedef struct LNode{
	double coef; //该项系数
	int exp;     //该项指数
	struct LNode *next;
}LNode,*LinkList;
```

使用链表头结点的exp变量存储链表长度。

采用链表指针数组 p[NUM] 存储各链表的头指针，数组元素初值为NULL，创建链表时从头开始查找空位置，将链表头指针存入第一个空位置中。

除法功能结果较为特殊，包含商多项式和余数多项式，故采用两个链表头结点组成的结构体类型存储除法的结果，类型定义如下：

```c
typedef struct DivResult{
	LinkList quotient;  //商多项式的头指针
	LinkList remainder; //余数多项式的头指针
}DivResult;
```

## 调试分析

各功能测试数据与问题解决如下：

1. 输入并创建多项式：

   1. 常规输入 1 2.5 3 6.1 8 -2.3 -1，没有出现问题。

   2. 输入负指数 -2 1 -1，问题：创建了含负指数结点。

      解决方法：在输入指数时添加合法性检测，只允许输入非负整数和-1（结束指令）。

      为插入、修改同样加入该检测。

   3. 输入重复指数 1 1 1 1 -1，问题：创建了指数重复的结点。

      解决方法：在输入指数后利用Locate函数检测是否重复，若重复则提示重新输入。

      为插入、修改同样加入该检测。

   4. 输入零系数 2 0 -1，问题：创建了零系数结点，空占位置。

      解决方法：在输入系数时检测是否为0，若为0则不创建结点。

      为插入、修改同样加入该检测。

2. 输出多项式：

   1. 常规输入，没有出现问题。

   2. 输入未创建的序号，问题：无法输出未创建的多项式。

      解决方法：为各功能输入序号时添加合法性检测，若当前位置元素为NULL，则提示未创建多项式。

   3. 输入仅含头结点多项式的序号，问题：无法输出仅含头结点的多项式。

      解决方法：在输出前增加判断，若仅含头结点，则输出0.000。

3. 求和：

   1. 常规输入，没有出现问题。

   2. 输入含有指数相同、系数互为相反数项的多项式，问题：结果出现零系数结点。

      解决方法：计算同指数项时检测系数结果是否为0，若为0则不创建结点。

      在求差中同样加入该检测。

4. 求差：没有出现问题。

5. 求值：没有出现问题。

6. 销毁：注意将数组p对应位置重置为NULL，没有出现问题。

7. 清空：没有出现问题。

8. 插入新项：已添加合法性检测，没有出现问题。

9. 删除项：利用Locate函数寻找删除项位置，该项不存在则打印未找到该项，没有出现问题。

10. 修改已有项：已添加合法性检测，没有出现问题。

11. 微分：

    1. 常规输入，没有出现问题。

    2. 输入负数阶数，问题：无法计算负阶数微分。

       解决方法：加入合法性检测。

       在乘方中同样加入该检测。

12. 不定积分：没有出现问题。

13. 定积分：没有出现问题。

14. 乘法：没有出现问题。

15. 乘方：没有出现问题。

16. 除法：

    1. 常规输入，问题：商多项式项数错误。

       解决方法：在计算过程中添加商多项式L->exp的自增。

    2. 被除数输入0，问题：结果错误。

       解决方法：加入合法性检测。

    3. 除数项数大于被除数项数，问题：结果错误。

       解决方法：加入合法性检测。

# 栈与队列的应用 实验报告

## 设计目标

在C语言环境下，利用栈与队列的相关操作，完成如下程序设计：

1. 简单的行文本编辑器

   建立一个空的文本编辑器，输入指令条数N，从输入文件中读入N条操作指令并执行。操作指令包括：

   Move k：将光标移动到第 k 个字符之后，如果 k=0 则将光标移到文本第一个字符之前。

   Insert n \n S：在光标后插入长度为 n 的字符串 S，光标位置不变，n≥1。

   Delete n：删除光标后的 n 个字符，光标位置不变，n≥1。  

   Rotate n：反转光标后的 n 个字符，光标位置不变，n≥1。  

   Get：输出此时程序内保存的文本。

   Prev：光标前移一个字符。

   Next：光标后移一个字符。

   对所有执行过的Get操作，将指定的内容写入输出文件。保证输入数据合法。

2. 括号配对检验

   输入表达式数目N，输入N个含有英文字母、数字、四则运算符、左右大中小括号的表达式，以'@'作为表达式的结束符，检查表达式中的左右大中小括号是否匹配，若匹配则输出"YES"，否则输出"NO"。

3. 银行业务模拟

   客户业务分为两种：第一种是申请从银行得到一笔资金，即取款或借款；第二种是向银行投入一笔资金，即存款或还款。银行有两个服务窗口，相应地有两个队列。客户到达银行后先排第一个队，处理每个客户业务时，如果属于第一种，且申请额超出银行现存资金总额而得不到满足，则立刻排入第二个队等候直至满足时才离开银行；否则业务处理完后立刻离开银行，每接待完一个第二种业务的客户，则顺序检查和处理(如果可能)第二个队列中的客户，对能满足的申请者予以满足，不能满足者重新排到第二个队列的队尾。任何时刻都只开一个窗口。假设检查不需要时间，营业时间结束时所有客户立即离开银行。设计上述银行业务的事件驱动模拟系统，通过模拟方法求出客户在银行内逗留的平均时间。

   第一行输入四个数 N、total、close_time、average_time，分别表示来银行的总人数、银行开始营业时拥有的款额、今天预计的营业时长和客户交易时长，之后的 N 行每行输入两个数 a、b，第一个数 a 为客户办理的款额，用负值和正值分别表示第一类和第二类业务。第二个数 b 为客户来到银行的时间。

   输出前 N 行分别对应每个客户的等候时长，第 N+1 行为客户等候平均时长（保留整数）。

## 括号配对检验

### 算法设计

读取表达式，遇到左括号则判断其优先级是否低于栈顶左括号的优先级（'(' < '[' < '{'）；如果是，则将该括号入栈；否则认为括号配对错误。遇到右括号则判断其是否与栈顶左括号匹配，若匹配则将左括号出栈；否则认为括号配对错误。在字符串读取完毕后，判断栈是否为空，若不为空则认为括号配对错误，否则括号配对正确。

## 银行业务模拟

### 算法设计

读取客户信息建立客户链表customers，建立事件链表，并将第一个客户的到达事件插入链表。直到事件链表空为止循环处理链表的第一个事件。全部处理完毕后，计算仍未得到处理的客户的等待时间（即一直等待到银行关门）。

# 二叉树的应用 实验报告

## 设计目标

在C++环境下，利用二叉树与Huffman树的相关操作，实现对任意文件的压缩处理，压缩之后的文件后缀名为huff，同时可以对所有后缀名为huff的压缩文件进行解压缩。

以1个字节为单位进行Huffman编码，要求可以完成压缩和解压的步骤，并且解压出来的文件没有任何损失。

## 算法设计

压缩文件时执行的操作： 

1. 读入文件，统计每种字节出现的次数（256种） 
2. 根据字节出现的次数创建Huffman树
3. 由Huffman树得到各字节的Huffman编码
4. 将压缩前文件类型、各字节出现的次数信息写入.huff文件
5. 读入文件，按字节将对应的编码写入.huff文件，缺位用0补齐

解压缩文件时执行的操作：

1. 读入.huff文件，根据文件开头的信息获得Huffman树 
2. 根据Huffman编码，将对应的字节写入文件

.huff文件结构为：

```c++
char FileType[FILE_TYPE_LENGTH] //源文件类型（,xxx） 
int FileSize //源文件大小（用于解压时确定是否解码结束） 
int count[256] //各字节按顺序的出现次数 
//Huffman码比特流 
```

基于贪心策略构建Huffman树的算法思想概括如下：

1. 给定n个权值作为n个叶子节点，构成初始的森林F={T1, T2, ..., Tn}，其中每棵二叉树Ti仅有一个权值为Wi的根节点。
2. 在森林F中选取两棵根节点权值最小的树作为左右子树，构造一棵新的二叉树，并置新二叉树根节点的权值为其左右子树上根节点的权值之和。
3. 从森林F中删除这两棵被选中的树，同时将新构造的二叉树加入到森林F中。
4. 重复步骤2和步骤3，直到森林F中只含一棵二叉树为止，即得到Huffman树。

### 数据结构

采用结构体数组FileByteInfo[256]存储各字节（无符号字符）的信息，下标即表示无符号字符的值，结构体定义为：

```c++
typedef struct Info{
	int count;	//无符号字符出现的次数
	char *HCode;	//字符对应的Huffman编码 
}Info;	//文件内无符号字符的信息
```

Huffman编码过程借助Huffman树实现，Huffman树采用顺序存储，其结点结构定义为：

```c++
typedef struct HTNode{
	unsigned char data;
	int weight;		//权重
	int lchild,rchild;
	int parent;
}HTNode,*HuffmanTree;
```

### 程序结构

各函数原型、功能与接口描述如下：


## 图的遍历

### 算法设计

深度优先遍历（DFS）采用递归算法，算法思想如下：

1. 初始化访问情况数组，从起始节点开始，将其标记为已访问，以避免重复访问。
2. 对于起始节点的每一个未访问的邻接节点，递归地调用深度优先遍历函数。这意味着每次找到一个新的邻接节点，就将其视为新的起始节点，继续深入探索它的邻接节点。
3. 当一个节点的所有邻接节点都被访问后，递归函数会返回上一层调用。回溯到上一层后，继续探索上一层节点的其他未访问邻接节点。
4. 当所有节点都被访问过，或者没有更多的邻接节点可以探索时，递归过程结束，从小到大对未访问结点再次进行如上操作。

广度优先遍历（BFS）借助队列实现，算法思想如下：

1. 初始化访问情况数组，从起始节点开始，将其标记为已访问，以避免重复访问。
2. 访问起始结点并将起始节点入队。
3. 直到队列为空为止循环：取队列头节点，从小到大访问其未访问的邻接节点，并将它们入队。
4. 循环结束，从小到大对未访问结点再次进行如上操作。

## 求通讯网的最小代价生成树

### 算法设计

Prim算法的基本思想是从一个起始顶点开始，逐步扩展生成树，直到包含所有顶点，算法思想如下：

1. 从连通无向网G中选择一个起始顶点u0，首先将它加入到集合T中。
2. 在集合T中的顶点u与不在T中的顶点v之间选择权值最小的边(u, v)，将顶点v加入到集合T中。
3. 重复步骤2，直到网络中的所有顶点都加入到生成树顶点集合T中为止。

Kruskal算法的基本思想是从边的角度出发，逐步构建最小生成树。算法思想如下：

1. 连通网G=（V，E），令最小生成树的初始状态为只有n个顶点而无边的非连通图T=（V，{}），即图中每个顶点自成一个连通分量。

2. 在E中选择权值最小的边，若该边依附的顶点落在T中不同的连通分量上（即不形成回路），则将此边加入到T中，否则舍去此边而选择下一条权值最小的边。

3. 重复上述步骤，直到T中所有顶点都在同一连通分量上为止。

## 铁路交通网的最短路径

### 算法设计

Dijkstra算法通过贪心策略，不断选择当前路径代价最小的节点进行更新，逐步扩展搜索范围，直到找到从源节点到所有可达节点的最短路径。算法描述如下：

1. 从起点开始，将起点的距离设置为0，其他节点的距离设置为无穷大，表示还未找到到达该节点的路径。

2. 从未访问的节点中，选择距离起点最近的节点，然后将其标记为已访问。

3. 对选中节点的所有邻居节点，检查从起点经过当前节点到达邻居节点的路径是否比当前已知最短路径更短，如果是，则更新邻居节点的最短路径值。

4. 重复步骤2和步骤3，直到所有节点都被访问，或剩余未访问节点与起点不连通。

5. 算法完成时，每个节点的最短路径长度都已确定。

测试文件中包括：关键字个数 n，关键字 key（这里我们认为关键字 key 就是哈希表中元素对应的哈希函数值），和除留余数法中的 p。要求按顺序输入关键字后，输出哈希表的相关信息，包括表的结构， 成功查找次数， 失败查找次数， 平均查找长度等信息。

## 算法设计

用除留余数法构建的哈希函数为：
$$
H(key)=key \mod p\text{ },\text{ }p\leq m
$$
此题中直接取哈希表长m=p。

线性探测法求得的第i个地址为：
$$
H_i=(H(key)+i)\mod m \text{, }i=1,2,\dots,k(k\leq m-1)
$$
拉链法直接采用链表，在同一地址上存储多个记录。

查找时直接利用关键字key求出地址H(key)，在HT1上利用线性探测寻找该记录，成功查找次数$S$即为线性探测的次数i，若遇到空位置说明查找失败，失败查找次数$U$即为空位置与初始位置的距离；在HT2上遍历该地址上的链表，成功查找次数$S$即为该记录在链表上的位置，遍历完仍未找到说明查找失败，失败查找次数$U$即为链表长度+1。

假设各记录被查找概率相等即可计算出ASL：
$$
ASL = \frac{1}{n}\sum^{n}_{i=1}S_i
$$

```c
		}
	}
	while(pa){

			exit(OVERFLOW);

double EvaluateList(LinkList L,double x){
	//代入x的值计算多项式的值并返回
	double sum = 0,pow;
	int i;
	LNode *p = L->next;
	while(p){
		pow = 1;
		for(i=0;i < p->exp;i++)
			pow *= x;
		sum += p->coef * pow;
		p = p->next;
	}
	return sum;
}

DivResult DivList(LinkList La,LinkList Lb){
	//输入被除式与除式，以结构体形式输出商和余数的头指针
	DivResult res;
	LinkList Mult=NULL,temp = NULL;
	LNode *pa = La->next,*pb = Lb->next;
	LNode *p = NULL,*q = NULL;
	res.quotient = (LinkList)malloc(sizeof(LNode));
	p = (LNode*)malloc(sizeof(LNode));
	if(p==NULL){
		printf("内存错误!\n");
		exit(OVERFLOW);
	}
	p->coef = pa->coef / pb->coef;
	p->exp = pa->exp - pb->exp;
	p->next = NULL;
	res.quotient->next = p;
	res.quotient->exp = 1;
	q = p;
	Mult = MultList(res.quotient,Lb);
	res.remainder = SubList(La,Mult);
	DestroyList(Mult),Mult = NULL;
	while(res.remainder->next != NULL && res.remainder->next->exp >= pb->exp){
		p = (LNode*)malloc(sizeof(LNode));
		if(p==NULL){
			printf("内存错误!\n");
			exit(OVERFLOW);
		}
		p->coef = res.remainder->next->coef / pb->coef;
		p->exp = res.remainder->next->exp - pb->exp;
		p->next = NULL;
		q->next = p;
		q = p;
		res.quotient->exp++;
		Mult = MultList(res.quotient,Lb);
		temp = res.remainder;
		res.remainder = SubList(La,Mult);
		DestroyList(temp);temp=NULL;
		DestroyList(Mult);Mult=NULL;
	}
	return res;
}
```

```c
#include <stdlib.h>
#define OK 1
#define ERROR 0

typedef int Status;

typedef struct SqStack{
	char *base;
	char *top;

int strcmp(char *A,char *B){
	char *pa = A,*pb = B;
	while(*pa && *pb){
		if(*pa > *pb) return 1;
		if(*pa < *pb) return -1;
		pa++;pb++;
	}
	if(*pa && !*pb) return 1;
	if(!*pa && *pb) return -1;
	if(!*pa && !*pb) return 0;
}//字符串比较函数 

int JudgeOpe(char *ope){
	if(strcmp(ope,"Get") == 0) return 1;
	else if(strcmp(ope,"Prev") == 0) return 2;
	else if(strcmp(ope,"Next") == 0) return 3;
	else if(strcmp(ope,"Move") == 0) return 4;
	else if(strcmp(ope,"Insert") == 0) return 5;
	else if(strcmp(ope,"Delete") == 0) return 6;
	else if(strcmp(ope,"Rotate") == 0) return 7;
	else return 0;
}//判定操作内容，转换为编号 

Status Move(SqStack *PText,SqStack *NText,int num){
	int i; 
	int ssp=PText->stacksize,ssn=NText->stacksize;
	if(num < ssp){
		//需移动位置在当前光标位置之前
		//将PText中的ssp-num个元素出栈，放入NText
		for(i=0;i<ssp-num;i++){
			*NText->top = *(PText->top-1);
			*(PText->top-1) = 0;
			NText->top--;PText->top--;
			NText->stacksize++;
			PText->stacksize--;	
		}
	}
	if(num > ssp){
		//需移动位置在当前光标位置之后 
		//将NText中的num-ssp个元素出栈，放入PText
		for(i=0;i<num-ssp;i++){
			*PText->top = *(NText->top+1);
			*(NText->top-1) = 0;
			NText->top++;PText->top++;
			NText->stacksize--;
			PText->stacksize++;	
		}
	}
	return OK; 
}

Status Prev(SqStack *PText,SqStack *NText){
	//将PText栈顶元素出栈，放入NText
	*NText->top = *(PText->top-1);
	NText->top--;PText->top--;
	NText->stacksize++;
	PText->stacksize--;	  
	return OK;
}

Status Next(SqStack *PText,SqStack *NText){
	//将NText栈顶元素出栈，放入PText
	*PText->top = *(NText->top+1);
	NText->top++;PText->top++;
	NText->stacksize--;
	PText->stacksize++;	
	return OK;
}

Status Insert(SqStack *PText,SqStack *NText,FILE *fp,int num){
	//从fp开始读入字符存入PText，再将光标移动至原处
	int i;
	int start = PText->stacksize;//记录原光标位置
	fgetc(fp); 
	for(i=0;i<num;i++){
		*PText->top = fgetc(fp);
		PText->top++;
```

```c
Status BracketCmp(char a,char b){
	//判定括号a是否比括号b优先级低
	int la,lb;
	if(a == '(') la = 1;
	if(a == '[') la = 2;
	if(a == '{') la = 3;
	if(b == '(') lb = 1;
	if(b == '[') lb = 2;
	if(b == '{') lb = 3;
	if(la < lb) return TRUE;
	else return FALSE;
} 

Status BracketPairing(char a,char b){
	//判定右括号a是否与左括号b配对
	int la,lb;
	if(a == ')') la = 1;
	if(a == ']') la = 2;
	if(a == '}') la = 3;
	if(b == '(') lb = 1;
	if(b == '[') lb = 2;
	if(b == '{') lb = 3;
	if(la == lb) return TRUE;
	else return FALSE; 
}

Status CheckBracket(char *text,SqStack S){
	//检查括号是否合法，若合法返回TRUE，否则FALSE
	char *pc = text;
	while(*pc != '@'){
		if(*pc == '(' || *pc == '[' || *pc == '{'){
			//字符为左括号 
			if(S->base == S->top || BracketCmp(*pc,GetTop(S))){
				//若栈空，或该括号比栈顶括号优先级低，入栈 
				Push(S,*pc);
			}
			else return FALSE;
			//若该括号不比栈顶括号优先级低，则错误 	
		}
		if(*pc == ')' || *pc == ']' || *pc == '}'){
			//字符为右括号
			if(S->base != S->top && BracketPairing(*pc,GetTop(S))){
				//若栈非空，且括号与栈顶左括号配对，出栈 
				Pop(S); 
			}
			else return FALSE;
			//若该括号不与栈顶括号配对或栈空，则错误 
		}
		pc++;
	}
	if(S->base != S->top)
	//若此时栈不为空，则错误 
		return FALSE;
	//完成所有括号入栈与出栈，括号合法 
	return TRUE; 
}
```

```c++
Status FindSmallest(HuffmanTree T,int i,int &s1,int &s2){
	//找到数组中权重最小且未被连接过的两个结点 
	int j = 1;
	while(T[j].parent) j++;
	s1 = j;
	for(j++;j < i;j++){
		if(T[j].weight < T[s1].weight && T[j].parent == 0) s1=j;
	}
	j=1;
	while(T[j].parent || j == s1) j++;
	s2 = j;
	for(j++;j < i;j++){
		if(T[j].weight < T[s2].weight && j!=s1 && T[j].parent == 0) s2=j;
	}
	return OK;
}

Status InitHuffmanTree(HuffmanTree &T,Info *FileByteInfo,int types){
	//建立Huffman树（顺序存储类型） 
	T = (HTNode*)malloc((sizeof(HTNode)*types*2));
	int i=1,j=0;
	int s1,s2;
	for(i=1;i<=2*types-1;i++){
		T[i].lchild = 0; T[i].rchild = 0;
		T[i].parent = 0;
	} 
	i=1;
	for(j=0;j<256;j++){
		if(FileByteInfo[j].count){
			T[i].data = j;
			T[i].weight = FileByteInfo[j].count;
			i++;
		}
	}//完成每个结点的输入
	for(i=types+1;i <= 2*types-1;i++){
		FindSmallest(T,i,s1,s2);
		//找到1到i-1中权重最小且未被连接过的两个结点
		T[i].lchild = s1;
		T[i].rchild = s2;
		T[i].weight = T[s1].weight+T[s2].weight;
		T[s1].parent = i;T[s2].parent = i;
		//完成连接 
	}//完成Huffman树的创建，最后一个节点为根
	return OK;
}

Status HuffmanCoding(HuffmanTree T,Info *FileByteInfo,int types){
	//根据Huffman树得到各字符的Huffman编码
	char *cd = (char*)malloc(sizeof(char)*types);
	//构建工作区cd
	cd[types-1] = '\0';
	int i;
	int c,f,start;
	for(i=1;i<=types;i++){
		start = types-1;
		c = i; f = T[i].parent;
		while(f){
			if(c == T[f].lchild){
				cd[--start] = '0';
			}
			else{
				cd[--start] = '1';
			}
			c = f;
			f = T[f].parent;
		}
		FileByteInfo[T[i].data].HCode = (char*)malloc(sizeof(char)*(types-start));
		strcpy(FileByteInfo[T[i].data].HCode,&cd[start]);
	}
	return OK; 
}

		if(ReadSize == 0){
			//读取完毕 
			break;
		}
		for(i=0;i<ReadSize;i++){
			code = FileByteInfo[ReadBuff[i]].HCode;
			while(*code){
				ch = ch << 1;
				if(*code == '1')
					ch = ch | 1;
				BitCount++;
				if(BitCount == 8){
					fputc(ch,NewFile);
					BitCount = 0;
				}
				code++;
			}
		}
	}
	if(BitCount != 0){
		//剩余不足1字节的编码未写入 
		ch = ch << (8-BitCount);
		fwrite(&ch,1,1,NewFile);
	}
	return OK; 
}

Status HuffmanDecoding(HuffmanTree T,int types,int FileSize,FILE *OriginFile,FILE *NewFile){
	//按Huffman编码还原源文件 
	unsigned char ReadBuff[1024];
	int ReadSize;
	int SizeCount = 0; 
	int i;
	int pt = 2*types - 1; //从根开始 
	unsigned char ch = 0;
	int BitCount = 0; 
	//每次读取1KB数据 
	while(1){
		ReadSize = fread(ReadBuff,1,1024,OriginFile);
		if(ReadSize == 0){
			//读取完毕 
			break;
		}
		for(i=0;i<ReadSize;i++){
			ch = ReadBuff[i];
			for(BitCount = 0;BitCount < 8;BitCount++){
				if(ch & 0x80){
					//读取位为1 
					pt = T[pt].rchild;
				}
				else{
					//读取位为0 
					pt = T[pt].lchild;
				}
				if(T[pt].lchild == 0 && T[pt].rchild == 0){
					//找到叶子结点，输出对应字节
					fputc(T[pt].data,NewFile);
					pt = 2*types - 1;
					//pt重置为根
					SizeCount++;
					if(SizeCount == FileSize){
						break;
					}
				}
				ch = ch << 1;
			}
		}
	}
	return OK;
}
```

```c++
	if(!Q.front->next)

	return OK;
}
```

```c++
	//返回树各边的长度之和 
	closedge *close = (closedge*)malloc(sizeof(closedge)*(G.vexnum+1));
	if(!close){
		perror("内存分配失败\n");
		exit(OVERFLOW); 
	}
	int i,j,k;
	int rightsum = 0;
	int min;
	for(i = 2;i <= G.vexnum;i++){
		close[i].adjvex = 1;
		close[i].lowcost = G.arcs[1][i];
	}	//初始化close数组

	for(i = 1;i < G.vexnum;i++){
		min = INFINITY;
		for(j = 1;j <= G.vexnum;j++){
			if(close[j].lowcost && close[j].lowcost < min){
				min = close[j].lowcost;
				k = j;
			}
		}
		//找到距离最短的下个顶点k 
		rightsum += G.arcs[close[k].adjvex][k];
		//增加总长
		close[k].lowcost = 0;
		//第k顶点并入已有树
		for(j = 1;j <= G.vexnum;j++){
			if(G.arcs[k][j] < close[j].lowcost){
				close[j].adjvex = k;
				close[j].lowcost = G.arcs[k][j];
			}
		} //更新close数组 
	}
	return rightsum;
}
```

```c++
int ShortestPath_DIJ(MGraph G,int start,int end){
	//用Dijkstra算法计算start到end的最短路径
	//返回最短路径的长度 
	int *distance = (int*)malloc(sizeof(int)*(G.vexnum+1));
	if(!distance){
		perror("内存分配失败\n");
		exit(OVERFLOW); 
	}	//存储到顶点的最短距离 
	int *final = (int*)malloc(sizeof(int)*(G.vexnum+1));
	if(!final){
		perror("内存分配失败\n");
		exit(OVERFLOW); 
	}	//存储是否经过顶点 
	int i,j,k;
	int min;
	for(i = 1;i <= G.vexnum;i++){
		distance[i] = G.arcs[start][i];
		final[i] = FALSE;
	}	//初始化 
	final[start] = TRUE;
	distance[start] = 0;
	for(i = 1;i < G.vexnum;i++){
		min = INFINITY;
		for(j = 1;j <= G.vexnum;j++){
			if(!final[j] && min > distance[j]){
				k = j;
				min = distance[j];
			}
		}
		//找到未经过的最近顶点k
		final[k] = TRUE;
		//标记经过k 
		for(j = 1;j <= G.vexnum;j++){
			if(!final[j] && distance[j] > min+G.arcs[k][j]){
				distance[j] = min+G.arcs[k][j];
			}
		}
		//更新未经过顶点最近距离 
	}
	//完成各顶点最近距离计算
	return distance[end]; 
}
```

```c++
#include<stdio.h>
#include<stdlib.h>
#define OK 1
#define ERROR -1
#define SUCCESS 1
#define UNSUCCESS 0
#define DUPLICATE -1
#define NULLKEY 2147483647
typedef int Status;

typedef struct CNode{
	int elem;
	struct CNode *next;
}CNode,*Chain;

typedef struct CHTable{
	Chain *ChainHash;
	int *ChainLength; 
	int count;
}CHashTable;

typedef struct HTable{
	int *elem;
	int *successtime;
	int *failtime; 
	int count;
}HashTable;

int tablesize,num;

int Hash(int x){
	//计算哈希地址
	return x % tablesize;
}

Status InitTable(HashTable &HT,CHashTable &CHT,int *data){
	int i,k,count = 0; 
	int address;
	CNode *p = NULL,*q = NULL;
	HT.elem = (int*)malloc(sizeof(int)*tablesize);
	HT.successtime = (int*)malloc(sizeof(int)*tablesize);
	HT.failtime = (int*)malloc(sizeof(int)*tablesize);
	HT.count = 0;
	CHT.ChainHash = (Chain*)malloc(sizeof(Chain)*tablesize);
	CHT.ChainLength = (int*)malloc(sizeof(int)*tablesize);
	CHT.count = 0;
	for(int i = 0;i < tablesize;i++){
		HT.elem[i] = NULLKEY; 
		HT.successtime[i] = 0;
		HT.failtime[i] = 0;
		CHT.ChainHash[i] = NULL;
		CHT.ChainLength[i] = 0;
	}
	//创建与初始化 
	for(i = 0;i < num;i++){
		count = 1;
		address = Hash(data[i]);
		//插入HT表内容 
		HT.count++;
		k = address;
		while(HT.elem[k] != NULLKEY){
			k = (k+1)%tablesize;
			count++;
		}
		HT.successtime[k] = count;
		//找到插入位置，记录成功查找次数 
		HT.elem[k] = data[i];
		//完成赋值
		//插入CHT表内容 
		CHT.count++; 
		p = (CNode*)malloc(sizeof(CNode));
		p->elem = data[i];
		p->next = NULL;
		//完成新结点创建 
		q = CHT.ChainHash[address];
		if(!q)
			CHT.ChainHash[address] = p;
		else{
			while(q->next && q->next->elem < data[i])
				q = q->next;
			p->next = q->next;
			q->next = p;
		}
		//按顺序插入结点
		CHT.ChainLength[address]++; 
	}
	for(i = 0;i < tablesize;i++){
		//计算失败查找次数
		count = 1;
		k = i;
		while(HT.elem[k] != NULLKEY){
			k = (k+1)%tablesize;
			count++;
		}
		HT.failtime[i] = count;
	} 
	return OK;
}
```

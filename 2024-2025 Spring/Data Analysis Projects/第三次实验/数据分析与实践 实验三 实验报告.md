#### Q2. 数据集存在很多缺失值，输出各列缺失值的个数，并删除数据集的最后一列。基于更新后的数据，展示哪一列的缺失值最多，哪些列没有缺失值。

统计各列缺失值个数时，采用对每列所有值应用`isna()`方法再求和的操作。

代码：

```python
col_names = df.columns   # 取所有列名，方便展示
for col in col_names:
    num_isnull = df[col].isna().sum()   # 统计缺失值个数
    print(f"列名: {col}, 缺失值数: {num_isnull}")
    
df.drop(col, axis=1, inplace=True)  # 删除最后一列

col_names = df.columns   # 更新列名
max = 0 # 记录缺失值最大值
col_no_null = []   # 记录无缺失值列名
col_max_null = []    # 记录缺失值最多的列名（可能存在多个）

for col in col_names:
    num_isnull = df[col].isna().sum()
    if num_isnull == 0:
        col_no_null.append(col)    # 无缺失值
    if num_isnull > max:
        max = num_isnull    # 更新最多缺失值

for col in col_names:
    num_isnull = df[col].isna().sum()
    if max == num_isnull:
        col_max_null.append(col)    # 缺失值最多的列

print(f"缺失值最多的列为: ")
for col in col_max_null:
    print(f"\t{col}")
print(f"无缺失值的列为: ")
for col in col_no_null:
    print(f"\t{col}")
```

#### Q3. 数据集的有些列在所有记录上均有相同取值，作为独立的一列是相对冗余的。请查找并输出这些列的名称和取值，并阐述这些列代表的含义，最后删除这些列。  

`nuique()`方法可以输出`dataframe`一列中的取值种类数，若`nuique()`返回值为1，则说明该列在所有记录上均有相同取值。采用字典`col_dup`存储列名和取值。

代码：

```python
col_dup = {}   # 记录冗余列名和取值
for col in col_names:
    if df[col].nunique() == 1:
        col_dup[col] = df[col].loc[0]   # 列中只有一种数值
    
print("冗余列：")
for col, value in col_dup.items():
    print(f"\t列名: {col}, 取值: {value}")

df.drop(columns=list(col_dup.keys()),inplace=True)	# 删除冗余列
```

结果：

```
冗余列：
	列名: CYC, 取值: 07MS
	列名: ADMINMODE, 取值: 2
```

查找`codebook.xlsx`文件，得到两列的含义分别为：

CYC: PISA Assessment Cycle (2 digits + 2 character Assessment type - MS/FT). 即记录评估类型与周期，取值均为07MS。

ADMINMODE: Mode of Respondent (1 Paper, 2 Computer, SYSTEM MISSING Missing). 即调查模式，均使用计算机调查。

#### Q5. 选取特征`STUBEHA`,`TEACHBEHA`,`EDUSHORT`,`STAFFSHORT`，展示它们的基本统计特征信息（平均值、标准差、四分位点、最小值、最大值、 Pearson相关系数矩阵）。

这些基本统计特征信息都可以使用`Dataframe`类的相关方法实现，包括：

- 均值：`mean()`
- 标准差：`std()`
- p分位数：`quantile(p)`（$ p \in [0,1]$）
- 最大值：`max()`
- 最小值：`min()`
- Pearson相关系数矩阵：`corr(method='pearson')`

代码：

```python
features = ["STUBEHA","TEACHBEHA","EDUSHORT","STAFFSHORT"]  # 需统计的特征

for col in features:
    print(f"列{col}的基本统计特征信息:")
    print(f"\t平均值: {df[col].mean()}")
    print(f"\t标准差: {df[col].std()}")
    print(f"\t25%分位数: {df[col].quantile(0.25)}")
    print(f"\t50%分位数: {df[col].quantile(0.5)}")
    print(f"\t75%分位数: {df[col].quantile(0.75)}")
    print(f"\t最大值: {df[col].max()}")
    print(f"\t最小值: {df[col].min()}")

pearson_matrix = df[features].corr(method="pearson")
print("Pearson相关系数矩阵：")
print(pearson_matrix)
```

### 任务4：特征`STUBEHA`,`TEACHBEHA`的总体分布参数估计

基于正态分布假设，对特征`STUBEHA`,`TEACHBEHA`的总体分布进行参数估计。

#### Q1. 假设特征`STUBEHA`和`TEACHBEHA`样本分别独立同分布于正态分布$ N(\mu_1,\sigma_1^2) ,N(\mu_2,\sigma_2^2) $，请分别求均值参数$\mu_1,\mu_2$和方差参数$ \sigma_1, \sigma_2 $的极大似然估计。作为总体均值和总体方差的估计，样本均值和样本方差均具有无偏性，请问上述极大似然估计是否也同样具有无偏性？

求最大似然估计：

样本独立同分布于正态分布 $ N(\mu,\sigma^2) $，密度函数为 $ f(x) = \frac{1}{\sqrt{2\pi}\sigma} \exp\{-\frac{(x-\mu)^2}{2\sigma^2}\}$

似然函数$ L(\mu, \sigma^2 ; X) = \prod^n_{k=1} f(x_k) $

取对数得到
$$
\begin{aligned}

  l(\mu, \sigma^2) &= \sum^n_{k=1} \ln f(x_k)  \\

  &= \sum^n_{k=1} (-\frac{\ln 2\pi}{2} - \ln \sigma - \frac{(x_k-\mu)^2}{2\sigma^2})  \\

  &= -\frac{n \ln 2\pi}{2} - n\ln \sigma - \frac{1}{2\sigma^2}\sum^n_{k=1}(x_k-\mu)

\end{aligned}
$$
求驻点，即
$$
\begin{aligned}

  \frac{\partial l}{\partial \mu} &= \frac{1}{2\sigma^2}\sum^n_{k=1} 2(x_k-\mu) = 0 \\

  \frac{\partial l}{\partial (\sigma^2)} &= -\frac{n}{2\sigma^2} + \frac{1}{2(\sigma^2)^2}\sum^n_{k=1} (x_k-\mu)^2 = 0

\end{aligned}
$$
解得
$$
\begin{aligned}

  \hat{\mu} &= \frac{1}{n} \sum^n_{k=1} x_k  = \overline{x} \\

  \hat{\sigma^2} &= \frac{1}{n} \sum^n_{k=1}(x_k - \overline{x})^2 = m_2

\end{aligned}
$$
计算最大似然估计的代码：

```python
stu_mu = df2["STUBEHA"].mean()
squared_diffs = (df2["STUBEHA"] - stu_mu) ** 2
stu_sigma2 = squared_diffs.mean()
print(f"STUBEHA的最大似然估计: mu = {stu_mu:.4f}, sigma^2 = {stu_sigma2:.4f}")

teach_mu = df2["TEACHBEHA"].mean()
squared_diffs = (df2["TEACHBEHA"] - stu_mu) ** 2
teach_sigma2 = squared_diffs.mean()
print(f"TEACHBEHA的最大似然估计: mu = {teach_mu:.4f}, sigma^2 = {teach_sigma2:.4f}")
```

结果：

```
STUBEHA的最大似然估计: mu = 0.0411, sigma^2 = 1.5289
TEACHBEHA的最大似然估计: mu = 0.1082, sigma^2 = 1.3458
```

验证无偏性：
$$
\begin{aligned}

  E\hat{\mu} &= E(\frac{1}{n} \sum^n_{k=1} x_k)  \\

​    &= \frac{1}{n} \sum^n_{k=1} \mu \\

​    &= \mu

\end{aligned}
$$
即$ \mu $是无偏的.
$$
\begin{aligned}

  E\hat{\sigma^2} &= E(\frac{1}{n} \sum^n_{k=1} (x_k-\overline{x})^2) \\

  &= \frac{1}{n} E(\sum^n_{k=1} x_k^2 + n\overline{x}^2 - 2\overline{x}\sum^n_{k=1} x_k)  \\

  &= \frac{1}{n} E(\sum^n_{k=1} x_k^2 + n\overline{x}^2 - 2n\overline{x}^2)  \\

  &= \frac{1}{n} E(\sum^n_{k=1} x_k^2 - n\overline{x}^2)  \\

\end{aligned}
$$
又
$$
\begin{aligned}

  E(\sum^n_{k=1} x_k^2) &= \sum^n_{k=1} E(x_k^2)  \\

  &= n(\mu^2 + \sigma^2)  \\

  E(\overline{x}^2) &= \frac{1}{n^2} E(\sum^n_{k=1} x_k)^2 \\

  &= \frac{1}{n^2} E(\sum^n_{k=1} x_k^2 + \sum_{i \neq j}x_i x_j) \\

  &= \frac{1}{n^2} (n(\mu^2+\sigma^2) + n(n-1)\mu^2)  \\

  &= \frac{1}{n^2}(n^2\mu^2 + n\sigma^2)

\end{aligned}
$$
故
$$
E\hat{\sigma^2} = \frac{n-1}{n} \sigma^2 \neq \sigma^2
$$
即 $ \sigma^2 $是有偏的.

#### Q3. 基于Q2所得结果，请仔细斟酌并叙述你所得到的结论。  

在本问题的成对t检验中，t值为
$$
t = \frac{\overline{d} - \mu_0}{s_d/\sqrt{n}}​
$$
其中$ \overline{d} $为每对`STUBEHA`与`TEACHBEHA`样本的差$ d $的均值，$ s_d $为$ d $的标准差，$ \mu_0 $为零假设中的平均值之差（本问题中$ \mu_0 = 0$）。在零假设成立的条件下，$ t $符合自由度为$ n - 1 $的t分布，$ n $为样本数。

p值的定义为“零假设成立的条件下，样本的t统计量取当前t值及更极端值”的概率，“极端值”是利于对立假设的。p值越小，越有把握拒绝原假设，进而肯定对立假设。在本问题中，单侧检验p值为
$$
p_{oneside} = P(T < t|H_0), T \sim t_{n-1}
$$
计算得到$p_{oneside} = 2.4682470543872093 \times 10 ^{-21}$

这个p值小于0.05，故在显著性水平$\alpha = 0.05$下，有足够的把握拒绝零假设。即可以认为`STUBEHA`的总体均值明显小于`TEACHBEHA`的总体均值。

#### Q4. 上述结论隐含了犯哪一类错误的可能？相应犯错概率是多少？

第一类错误：在原假设成立的情况下拒绝了原假设（统计量落在拒绝域内）。

第二类错误：在对立假设成立的情况下接受了原假设（统计量不落在拒绝域内）。

由于结论拒绝了原假设，所以只会犯第一类错误。

在显著性水平$ \alpha = 0.05 $下，上述结论犯第一类错误的概率为：
$$
P(\text{第一类错误}) = P(T < t_\alpha| H_0) = \alpha = 0.05
$$
